## บันทึก

- การตรวจสอบดำเนินการโดยไม่เข้าถึง git เพื่อตรวจสอบหมายเลขสาขา; จึงสมมติหมายเลขฟีเจอร์ = 1 และ short-name = `create-personal-page` เนื่องจากไม่ได้รันคำสั่งตรวจสอบ remote/local ภายในเซสชันนี้
